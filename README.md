# Orma

Orma is your friendly database secretary. Whatever you need to query, she will get it for you.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     orma:
       github: sbsoftware/orma
   ```

2. Run `shards install`
